Build
-----
The relevant services are:
- database
- frontend
- vcs-proxy


```
$ cd docker
$ docker-compose build
```

Run
---
You can run all services with:
```
$ cd docker
$ docker-compose up
```

More
---
If you need to insert data into the database please refer to the commands available with:

```
$ cd docker
$ ./docker-admin.sh
```
